# Iris-Flower-Classification
This project demonstrates a complete machine learning pipeline using Python and Scikit-learn. It features a K-Nearest Neighbors (KNN) model to classify custom datasets (CSV/Excel). Includes automated data processing, Exploratory Data Analysis (EDA) with Seaborn pairplots, and a visual performance evaluation using dynamic confusion matrix heatmaps.


# K-Nearest Neighbors (KNN) Data Classifier 

## Project Overview
This project demonstrates a complete machine learning pipeline using Python and Scikit-learn. It uses a Supervised Learning algorithm (K-Nearest Neighbors) to classify data into distinct categories based on multiple input features. 

## Key Features
* **Exploratory Data Analysis (EDA):** Generates pairplots to visually map the relationships between different data features.
* **Automated Data Processing:** Capable of reading both `.csv` and `.xls/.xlsx` formats dynamically.
* **Model Training & Evaluation:** Splits data into training/testing sets and evaluates the model using Accuracy Scores and Classification Reports.
* **Visual Performance Metrics:** Outputs a generated Confusion Matrix heatmap to easily identify where the model predicts correctly versus where it gets confused.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, Seaborn, Scikit-learn

## How to Run
1. Clone this repository.
2. Ensure you have the required libraries installed (`pip install pandas matplotlib seaborn scikit-learn`).
3. Run the script and upload your dataset when prompted.
